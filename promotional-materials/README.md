## 🎨 使用指南

### 颜色主题
- **龙族**: 红色/橙色 (#FF4500, #FF6B35)
- **人类**: 蓝色/青色 (#00BFFF, #4A90E2)

### 使用场景
1. **社交媒体**: 使用 posters/ 中的海报
2. **网站**: 使用 banners/ 中的横幅
3. **视频**: 使用 screenshots/ 作为B-roll素材
4. **文档**: 使用 characters/ 展示角色特性
